# 👩‍💻 Portfólio Pessoal | Aline Marques

Bem-vindo ao repositório do meu portfólio pessoal! Este site foi desenvolvido para apresentar minha trajetória, formações acadêmicas, habilidades técnicas e os principais projetos de software e web que desenvolvi.  
🔗 **Acesse o portfólio online:** [https://rques.github.io/Meu-Portifolio/](https://rques.github.io/Meu-Portifolio/)

## 🎯 Objetivo do Projeto
Atuar como um currículo interativo, moderno e digital. O portfólio demonstra minhas competências em tecnologia e centraliza meus contatos profissionais, além de disponibilizar meu currículo em formato PDF para download direto.

## 🚀 Estrutura e Seções
O site é do tipo *Single Page Application* (página única) e está dividido nas seguintes seções:
* **Sobre Mim:** Apresentação pessoal, hobbies (música, livros e videogames) e objetivo profissional.
* **Experiência Profissional:** Histórico de atuação, como a experiência de Jovem Aprendiz em Administração.
* **Formação Acadêmica:** Cursos técnicos e graduação (Facens, SENAI e ETEC).
* **Projetos:** Demonstração prática de desenvolvimento, com links diretos para:
  * **ServMaster:** Sistema web focado na gestão de manutenção.
  * **Agenda de Controle Comercial (Doçura Gelada):** Sistema de registro de vendas e relatórios de renda.
  * **EcoMapa:** Plataforma interativa voltada à conscientização ambiental e reciclagem.
* **Habilidades e Tecnologias:** Destaque para HTML, CSS, JavaScript, VBA/Excel, Power BI, SAP (MM), entre outros.

## 🛠️ Tecnologias Utilizadas no Desenvolvimento
Este portfólio foi construído do zero utilizando as seguintes tecnologias web:
* **HTML:** Estruturação semântica e acessível.
* **CSS:** Estilização moderna com paleta *Dark Mode*, utilizando **Flexbox** e **CSS Grid** para responsividade.
* **Google Fonts:** Utilização da fonte 'Inter' para um design limpo e legível.
